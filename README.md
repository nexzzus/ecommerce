# La Tienda Online API es un sistema backend desarrollado para gestionar el funcionamiento de una tienda virtual.

# El sistema permite administrar usuarios, productos, categorías y pedidos, así como registrar los pagos asociados a cada compra y aplicar descuentos.

# La aplicación expone una API REST construida con FastAPI, conectada a una base de datos PostgreSQL en la nube (Neon), e incluye un menú por consola que permite realizar operaciones CRUD consumiendo la API.*/
